パフォーマンス比較

「テキストが何行になるか、高さは何pxか」を知る2つの方法を比較します。

DOM計測（従来）

非表示のdivにテキストを入れて offsetHeight を読む。
ブラウザは毎回「レイアウトリフロー」（ページ全体の配置再計算）を強制実行するため、500回繰り返すと重い。

Pretext（Canvas計測）

prepare() でCanvas APIを使い文字幅を事前キャッシュ（初回のみ）。
layout() はキャッシュ済みの数値で足し算するだけ。DOMに一切触らないので桁違いに速い。

import { prepare, layout } from "@chenglou/pretext"

// Pretext: 2ステップで高さ取得
const prepared = prepare(text, '16px "Noto Sans JP"')  // 初回のみ
const { height, lineCount } = layout(prepared, 300, 28) // 算術計算のみ
