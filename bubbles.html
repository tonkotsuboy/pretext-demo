チャットバブル — fit-content比較

CSSの width: fit-content は最長行に合わせるだけ。Pretextは行数を維持したまま最小幅に縮める（shrinkwrap）。

import { prepare, layout } from "@chenglou/pretext"

// 行数を維持したまま最小幅を二分探索
const base = layout(prepared, maxWidth, 24)
let lo = 60, hi = maxWidth, bestW = maxWidth
while (lo <= hi) {
  const mid = (lo + hi) >> 1
  layout(prepared, mid, 24).lineCount <= base.lineCount
    ? (bestW = mid, hi = mid - 1) : (lo = mid + 1)
}
